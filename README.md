Retail Sales Data Analysis Project

Project Overview:
This project analyzes a retail sales dataset to uncover insights related to sales performance, profitability, discount impact, regional trends, and product-level performance.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Key Insights:
- Technology category is the main driver of both sales and profit.
- Higher discounts are associated with lower profitability.
- The West region achieves the highest sales and profit.
- A small number of products generate most of the revenue.
- Some products consistently generate losses.

Conclusion:
This project demonstrates how data analysis can support business decision-making using real-world retail data.
